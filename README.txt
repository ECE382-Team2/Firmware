INSTALLING PSoC CREATOR

1. Visit this link to install PSoC Creator: https://community.infineon.com/t5/PSoC-Creator-Designer/Download-PSOC-Creator/td-p/343035

   If after installing, prompted to install Microsoft .NET Framework 3.5, hit the Windows button and then type "Turn Windows Features on or off". 
   Select the checkbox next to ".NET Framework 3.5 (includes .NET 2.0 and 3.0)", and click OK. Windows will then attempt to download the necessary 
   files from Windows Update to complete the installation (you may have to click a button to start this action), which may require a system restart.

3. Rerun the executable. 

4. Under the drop down menu, select "Typical" Installation. 

4. Click "Register Later"
